자바스크립트의 배열은 유동배열 : 크기, 데이터 변경이 가능

  • const myArray = [1,2,`a`,`b`,3];
  • 배열의 데이터 타입은 같게 만드는게 룰
  • myArray=[]; 초기값이 없는 빈 배열을 만든 후 데이터를 추가하여 사용
  • index를 가짐 => 0번지부터 시작
  • length(길이) : 총길이 확인가능
  • for of : 배열안의 내용을 탐색할 때 사용 [ ] for(let value of Array){ 코드; }
  • for in : 객체 안의 내용을 탐색할 때 사용 { } for(let value in Object){ 코드; }
