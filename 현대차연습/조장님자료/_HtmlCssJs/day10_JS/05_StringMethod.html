가장 많이 다루는 데이터 타입 : String

  • 문자열 데이터는 index(0부터 시작)를 가지고 있음.
  • 문자열 값 내에서 개행(br태그)하는 것을 추천하지 않는다.
  • new String() : 객체 타입으로 선언은 가능 / 추천 하지 않는다.
  • length : 문자열의 총길이를 나타낼때 사용
  • 문자열 관련 메소드 : java에서 사용하는 String 메소드와 매우 유사
  • 추출 관련 메소드 : substring(), substr(), slice()
  • 문자 대체 메소드 : replace(old, new) : 첫번째 매칭되는 문자만, 대소문자 구분
  • trim() : 양쪽 공백 제거
  • [앞쪽] padStart(길이, 채울문자) / [뒷쪽] padEnd(길이, 채울문자) ex) 주민등록번호 뒤에 가리는 용도, 주소 가릴때, 익명성이 필요할때
  • charAt(index) : 해당 index 문자 추출
  • indexOf("str") : 해당 문자열의 위치 반환
  • startWith("str") / endWith("str") : 해당 문자열이 앞 / 뒤에 존재하는지 확인하는 메소드 return 타입 => true / false
