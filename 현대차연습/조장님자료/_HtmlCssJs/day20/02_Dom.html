DOM(document Object Model) : 문서(html) 객체 모델

  • `' '` 하위 태그는 각각의 기능(method), 속성(properties)을 가지는 문서 객체이다.
  • 엘리먼츠 가져오기 : teg element로 가져오는 메서드 byId, byClassNAme, ByTagName
  • document.getElementById('idName') : 단일 엘리먼츠 리턴
  • document.getElementsByClassName('className') : 복수 엘리먼츠 리턴 => HTML collection 타입으로 리턴
  • document.getElementsByTagName('tagName') : 복수 엘리먼츠 리턴 => HTML collection 타입으로 리턴
