변수나 상수에 할당 할 수 없는 값의 종류

  • 숫자 : Number => 정수, 실수, NaN(Not a Number : 숫자가 아님)
  • 문자 : String => " ", ' ', ` ` 감싼 값들
  • 불리언 : Boolean = true / false
  • 객체 : Object => 1개 이상의 데이터가 "특정구조"를 가지고 있는 형태 { 키 : 값 } () : 아무것도 없는 사실도 1개의 객체로 취급 (없어도 Null로 뜨지않는다.)
  • 배열 : Array => [1,2,3,4..........] 형태로 이루어진 데이터 집합 배열의 특징 : 순서, 길이가 있음
  • 알 수 없는 값 : Null => 값이 있긴한데 파악이 안됨
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터 타입을 확인하는 명령어 : typeof 변수 / 값

    • 변수의 관리

    1. 기본 자료형 : 값이 메모리의 주소값이 가르키는 곳에 직접 할당 number, Boolean, String(참조자료형이지만 사용 편의상 같이 사용)
    2. 참조 자료형(reference Type) : 값이 저장되어 있는 주소를 할당 String, Object, Function, Null, Nudefinded
    3. == 같다(값만 비교), != 같지않다. === 같다(값과 타입이 같은지 비교) !== 같지않다.(값과 타입이 같지않은지 비교)
