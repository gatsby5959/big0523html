객체 : 식별할 수 있는 것을 상세 특징으로 표현하려는 형식

  • count로 선언하는 것이 일반적 : 그래야 객체가 변화되는것을 막을수 있음, 객체의 지속성을 유지하고 확보하기 위함
  • { } 속성명 : 속성값, 속성명: 속성값, ... 표현
  • 같은 객체 내에서 속성명은 중복되면 안됨. (고유 키)
  • 같은 객체내에서 속성명은 중복되면 안됨. (고유 키)
  • 객체 내부의 속성 값으로 사용될 수 있는 데이터의 타입은 제한이 없다.
  • 객체의 속성 값으로 익명함수를 정의 => 그 함수를 메서드라고 부른다. 메서드 이름은 속성명이 됨
