함수(function) : 특정 기능을 수행하기 위한 코드 - 호출되어야만 작동 자바는 클래스안에서 메서드가 생성 / 자바 스크립트는 어디서든 function 생성가능.

  • function 이라는 키워드를 사용하여 선언
  • function 함수명(매개변수){로직작성}
  • function 함수명 : 기능을 함축하는 의미로 작성
  • 파라미터(매개변수) 선택적으로 작성 (없어도 ()는 있어야 함.)
  • 자바스크립트에서는 파라미터 타입이 자유도가 높아서 다양한 형태로 사용가능
  • return 함수 실행 종료 역할 동시에 특정 결과값을 호출한 위치로 변환
